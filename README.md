# chant-counter
A simple, responsive, and interactive web app to help track your chanting or repetition practices like mantras, affirmations, or breath counting.
**Features:**
Custom Target Count: Enter any number or pick from presets (27, 54, 108).
**Auto Timer:** Automatically tracks elapsed time.
**Tap or Use Keyboard:** Tap the button or press Space/Enter to increase the count.
**Sound Feedback:** Audio beep per count, and a triple chime when you reach your target.
**Statistics Panel:** View average time per chant and chants per minute after completion.
**Minimal UI:** Clean, dark-themed interface with focus on usability.
**Technologies Used:**
HTML5
CSS3 (modern, responsive design)
JavaScript (vanilla) for functionality and audio
How to Use
Open the app in a browser.
Enter your chant target or select a preset.
Tap the button (or press Space/Enter) to start counting.
Once you've reached the goal, view your session stats and start again if needed!
